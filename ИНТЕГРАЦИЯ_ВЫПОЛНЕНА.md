## 4. Что сделать тебе

1. **Сервер бота**
   - Обновить файлы: `database.py`, `services/yookassa_service.py`, `config.py`, `keyboards/main_menu.py`, `.env.example`.
   - При первом запуске таблица `test_results` создастся сама (миграция в `init_db`).
   - Перезапустить бота (например `sudo systemctl restart your-bot-service`).
   - При необходимости добавить в `.env`: `MINI_APP_PATH_TO_SELF_URL=...` и/или проверить `MINI_APP_BACKEND_URL`.

2. **Vercel (обязательно для работы в боте)**
   - Сейчас Production URL может отдавать 404 (DEPLOYMENT_NOT_FOUND), пока нет успешного деплоя.
   - Открой https://vercel.com/ozharov164-9998s-projects/path-to-self-miniapp → **Deployments**. Если есть последний деплой — нажми у него **⋯** → **Redeploy**. Если деплоев нет — нажми **Deploy** (из ветки main). Дождись статуса **Ready**.
   - В **Settings** → **Environment Variables** проверь: `VITE_BOT_BACKEND_URL` = `http://217.114.11.97:8080`. Если менял — после этого снова **Redeploy**.
   - После успешного деплоя https://path-to-self-miniapp-ozharov164-9998s-projects.vercel.app должен отдавать 200 и открываться Mini App.

3. **Проверка**
   - В Telegram открыть бота → в меню должна быть кнопка **«🌱 Путь к Себе»**.
   - Нажать → открывается Mini App (Vercel).
   - Пройти тест, сохранить результат → в боте и в истории Mini App результат отображается.

После этого цепочка «бот → Mini App → бэкенд» работает полностью.

## Инструкция по тесту в Telegram

1. Открой **@CozyReset_bot** в Telegram.
2. В меню нажми **«🌱 Путь к Себе»**.
3. Должно открыться Mini App (3D-сад, дашборд).
4. Зайди в **Каталог тестов** → выбери любой тест → **Пройти**.
5. Ответь на вопросы (шкала 1–10), дойди до конца.
6. После сохранения увидишь «Результат сохранён ✅ Перейди в бота» и кнопку **«Открыть бота»**.
7. Вернись в бота, снова открой «🌱 Путь к Себе» → **История** — там должен быть сохранённый результат. Нажми **«Смотреть результат»** и проверь отображение.
8. Если что-то не открывается или 401 — открой приложение заново из чата с ботом (re-init выполнится автоматически).
